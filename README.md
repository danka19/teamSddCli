`teamSddCli` — локальный OpenSpec/Markdown-first process toolkit для
управляемой командной работы над изменениями. Он помогает понять следующий
шаг, роль, обязательные evidence и границу человеческого решения. AI может
объяснять и готовить разрешённую работу, но не подтверждает classification,
DoR/DoD, риск, merge, release или archive за человека.

Подробное описание проекта, архитектурных границ и текущего состояния находится
в [документации проекта](docs/README.md). Для первого знакомства начинайте с
FAQ, а не с внутренних Python-скриптов.
