# Статус возможностей простым языком

Канонический источник: [roadmap](../ROADMAP.md) и [Phase 3](../phases/PHASE_3_GUIDED_ROLE_AND_ANALYTICS_VERTICAL_SLICE.md).

| Состояние | Что это значит |
| --- | --- |
| Доступно сейчас | Локальный `sdd`, setup с подтверждением, ситуационный start/next, проверяемые гейты и read-only/preparation маршруты. |
| Запланировано | Корпоративная адаптация, наблюдаемый pilot и расширение автоматизации только через новые принятые изменения. |
| Намеренно заблокировано | Внешние мутации через `sdd run`, автопринятие AI, публикация, Jira-задачи, QA/AT-предложения и role inbox. |

Этот статус описывает границы использования, а не заменяет roadmap и OpenSpec lifecycle.
