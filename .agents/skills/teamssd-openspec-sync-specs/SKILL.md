# OpenSpec Sync Specs

Read the active change, every delta operation and the target living spec.
Apply additions, full modifications, removals and renames without losing
unmentioned scenarios. Preserve requirement headings and four-hash scenario
headings.

Run strict local OpenSpec validation and review the resulting diff. Do not
infer acceptance, archive the change or interact with external systems.
