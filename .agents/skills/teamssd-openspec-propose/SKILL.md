# OpenSpec Propose

Derive a kebab-case change ID and one primary roadmap phase. Use the local
OpenSpec CLI to create the change and obtain artifact instructions.

Create proposal, design when needed, one delta spec per affected capability and
trackable tasks. Keep proposal phase metadata and the roadmap inverse row
identical. Run strict OpenSpec and roadmap consistency checks.

Do not implement, publish or access external services in this workflow.
