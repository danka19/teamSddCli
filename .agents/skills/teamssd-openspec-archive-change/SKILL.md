# OpenSpec Archive Change

Require explicit human acceptance, completed tasks, verification evidence and
consistent lifecycle metadata. Sync reviewed deltas when required, archive
through the local OpenSpec CLI, update roadmap/docs and run strict validation.

Archive is a local state transition, not proof of delivery. Do not create a
remote PR, push, publish or merge. Report the exact external integration step
for a human or separately authorized workflow.
