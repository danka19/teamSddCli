# OpenSpec Explore

Read relevant local specs, active changes, code and docs. Separate facts,
inferences, proposals and human decisions. Ask focused questions and compare
alternatives without changing canonical files unless the human explicitly asks
to record a decision.

Exploration may end with a recommendation, a clarified question or a proposed
change. It never implies acceptance or implementation.

Do not browse the web or query external systems.
