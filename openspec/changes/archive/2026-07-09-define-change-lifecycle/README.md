# define-change-lifecycle

Draft OpenSpec proposal for the SDD change lifecycle contract.

This change defines proposed behavior for change package statuses, allowed transitions, Spec PR and archive gates, and ownership boundaries between deterministic checks, AI assistance, and human approval.

Status: archived. The human Option A gate accepted this change on 2026-07-09, and its requirements were promoted into `openspec/specs/change-lifecycle/spec.md`.
