# Evidence Audit

Define scope, criteria and authoritative local sources. Record each finding
with severity, exact evidence, affected requirement, consequence and proposed
remediation or human decision.

Separate verified facts from inference and missing evidence. Store substantial
audits as dated files under `docs/audits/` and update the project map. Absence
of evidence is not proof of success.

Do not query external sources or mutate external systems.
