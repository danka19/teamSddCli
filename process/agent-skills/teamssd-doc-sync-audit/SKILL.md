# Documentation Sync Audit

Compare `AGENTS.md`, roadmap, file structure, current audit, context,
verification checklist, relevant OpenSpec artifacts and changed implementation.
Identify stale status, missing files, duplicated rules and contradictions.

Correct unambiguous derived documentation. Route behavior, contract, lifecycle
or human-decision changes through their canonical artifact. Use
`teamssd-phase-status-audit` for phase state and end with
`teamssd-session-report`.

Use repository evidence only.
