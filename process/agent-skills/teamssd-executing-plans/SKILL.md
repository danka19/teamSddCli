# Executing Plans

Read and critically review the plan before editing. If it is actionable, track
its tasks, execute them in order and run each specified local verification.
Stop on ambiguity, repeated failure or a required human decision.

Preserve unrelated work and update plan/OpenSpec status only from evidence.
Finish with documentation reconciliation, local verification and
`teamssd-session-report`.

Never install packages, contact external systems, push, create a PR or deploy.
