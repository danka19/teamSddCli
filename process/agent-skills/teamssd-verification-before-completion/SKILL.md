# Verification Before Completion

For every intended claim identify the local command or manual inspection that
proves it. Run it fresh, read the full result and state the actual outcome.
Partial checks do not prove broader claims.

If a required check was waived or cannot run, report the exact reason and
residual risk. Never infer correctness from edited files or confidence.

Verification stays local and does not authorize network or external mutation.
