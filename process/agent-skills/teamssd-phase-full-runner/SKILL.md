# Phase Full Runner

Verify prerequisite phases, active plan, OpenSpec ownership and explicit
statuses. Execute work items in dependency order; use
`teamssd-subagent-driven-development` only for declared independent scopes.

Each item needs focused evidence and status updates. Close no phase until all
children and human gates are satisfied. Finish with consolidated documentation,
status audit, roadmap/OpenSpec validation and a full session report.

All operations remain local. Stop before remote Git, PR creation, publication
or deployment.
