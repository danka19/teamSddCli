# Phase Planner

Read the full project read order, relevant phase template, accepted specs,
active changes, audit evidence and implementation state. Run
`teamssd-phase-status-audit` and `teamssd-roadmap-openspec-validator`.

Plan from reconciled evidence, not `docs/ROADMAP.md` alone. Give every work
item dependencies, owned files, OpenSpec mapping, acceptance evidence, manual
gates, risks and one status. Preserve human decisions and phase handoff gates.

Keep all work local. External actions become explicit future human steps.
